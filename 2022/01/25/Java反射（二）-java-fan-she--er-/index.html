Java反射（二）

Created2022-01-25|Updated2022-01-25
|Post View:

概念

反射是一种机制，利用该机制可以在程序运行过程中对类进行解剖并操作类中的所有成员(成员变量，成员方法，构造方法)

使用反射操作类成员的前提

要获得该类字节码文件对象，就是Class对象

Class对象的获取方式

  • 方式1：通过类名.class获得
  • 方式2：通过对象名.getClass()方法获得
  • 方式3：通过Class类的静态方法获得： static Class forName(“类全名”)

    注意： 每一个类的Class对象都只有一个
